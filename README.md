# Color guessing game

This is an app that can:

* randomly display a RGB number
* 1 correct colors for the RGB and 5 incorrect colors for the RGB
* be responsive

## Live example

* https://myosubug.github.io/color-gussing-game/

## Prerequisites

* any web browser

## Installing

No installation is required.

## Built With

* Vanilla JS
* html
* css
